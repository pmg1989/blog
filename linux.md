### nohup yarn run start &
* 原生启动后台服务
    * 启动：nohup yarn run start &
    * 查看：jobs
    * 停止：fg [jobs的编号]

* 常用命令

    * which node : 用来查找一个命令的绝对路径
    * rm -rf dir : 强制删除dir目录
    * cp file1 file2 : cp [ 来源文件 ] [目的文件]
    * cp -r dir1 dir2 : cp [选项] [ 来源目录 ] [目的目录]
    * mv source dist : 移动或者重命名文件/目录
